Auth & Middleware Flow

Request Flow
Request → proxy.ts (Clerk middleware) → route check
/portaal/* → check: is Clerk user + heeft "klant" rol → ✓ door | ✗ redirect /portaal/inloggen
/dashboard/* → check: is Clerk user + heeft medewerker/admin rol → ✓ door | ✗ redirect /sign-in
/public/* → geen auth check, altijd door
